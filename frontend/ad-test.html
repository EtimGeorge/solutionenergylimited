SEESL Website - Ad Parameter Test Links

This page contains sample links that simulate ad URLs with various parameters for testing the SEESL website's parameter handling and lead tracking capabilities. These can be adapted for social media, email campaigns, or other marketing efforts.

Note: In real-world campaigns, replace "index.html", "about.html", etc., with your full domain and page path (e.g., "https://www.solutionenergylimited.com/about.html"). The parameters help track lead sources and can pre-fill forms or customize user experience.

Parameter Reference

Parameter Description Example Values Scope
source / utm_source The platform or origin of the traffic facebook, linkedin, google, newsletter, partner_site_X All Pages
campaign / utm_campaign Marketing campaign identifier brand_awareness_q3, engineering_services_promo, iso9001_nigeria All Pages
utm_medium Marketing medium (UTM format) cpc, social_paid, email, referral All Pages
service_interest / service The specific service the user is interested in (can pre-fill forms) engineering, procurement, asset_management, iso_certification, iso_training, iso_audit Contact Page, Service Pages
form Pre-select form type (e.g., for ISO contact forms) nigeria, international ISO Contact Forms
ad_id / offer_code Unique ad or offer identifier for specific tracking vt2025, SEESL20OFF, fb_ad_eng_001 All Pages
ref Referral source identifier partner_site_X, affiliate_Y All Pages
How to use in real campaigns:
When setting up actual ads, ensure the base URL (e.g., "https://www.solutionenergylimited.com/") is correct, followed by the page path and then the parameters (starting with "?"). For subsequent parameters, use "&".
